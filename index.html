01. 個人受注案件

東柏太鼓教室 公式Webサイト 新規制作

旧サイトの閉鎖に伴い、公式Webサイトをゼロから再構築した個人案件です。前職でのディレクション経験を活かし、「今、本当に発信すべき情報は何か」をクライアントと一緒に整理する要件定義から担当しています。お子様からご年配の方まで幅広い層が訪れるため、迷わず情報にたどり着ける分かりやすい設計を心がけました。また、手戻りを防ぐための綿密な進行管理も徹底しています。
デザイン面では、太鼓の「和」の雰囲気が直感的に伝わるよう、スクロールに合わせて要素がふんわりと現れるような、さりげないアニメーションを取り入れ、教室の魅力が引き立つように工夫しました。
使用ツール：HTML / CSS / JS / Photoshop / VS Code

02. Group Work：コンペ採用・実運用案件

畑英会話スクール Webサイトリニューアル

職業訓練校のグループワークで取り組んだリニューアル案件です。納期が6日間と短く、途中でメンバーが減ってしまうという厳しい状況でした。この課題を乗り越えるため、生成AIをコーディングの補助として導入し、ベース部分の構築を大幅に効率化しました。
そして、AIの活用で生み出せた時間を、スマホ表示のレイアウト調整や、ユーザー視点での使いやすさの改善、SEOの基礎設定といった「手作業が必要な細部の作り込み」に注いでいます。ツールを適材適所で使い分けながら最後はチームで協力して完成させ、結果として4チームによるコンペで1位に選ばれました。現在も公式Webサイトとして実運用していただいています。
使用ツール：HTML / CSS / JS / Photoshop / VS Code / Figma

03. Group Work：マーケティング課題

イベント集客用LP（架空課題）

グループワークで取り組んだ、架空のイベント集客用ランディングページです。単に作るだけでなく、売上目標を達成するためのマーケティング分析（3CやSTP分析など）から話し合いに参加しました。限られた制作期間の中でクオリティを担保するため、国内のスマホシェアや検証にかかる時間を考慮し、ターゲットの基準をiPhoneのサイズに絞り込むという判断をしています。
制作中はFigmaを使ってデザイナーと細かく連携を取り、実装者の視点からUIの提案を行いました。また、スマートフォンでの表示速度や閲覧のしやすさを最優先に考え、過剰なアニメーション（JavaScript）はあえて省き、「情報をしっかり読ませる」ことに集中できるシンプルな実装を心がけました。
使用ツール：HTML / CSS / JS / Photoshop / VS Code / Figma

About Me & Skills

プロフィールとスキルセット

「作る」ことと「進める」こと。領域を限定せず、臨機応変に動ける制作者を目指して。

前職のWeb制作会社では、デザイナーとしてWebサイトの新規制作やECサイトのバナー・商品画像作成を行うと同時に、ディレクターとしてクライアントとの打ち合わせやスケジュールの進行管理も担当していました。また、売上・在庫管理からお客様の電話・メール対応まで、Web制作と運用に関わる幅広い業務を経験しました。
その後、一身上の都合によりWeb業界から離れていた期間がありました。しかし、「もう一度この業界で本気で働きたい」という強い思いから、直近の6ヶ月間、職業訓練校に通いました。そこではFigmaなどのデザインツールや現在のコーディング手法に加え、Premiere Proを使った動画編集の基礎など、今のWeb業界で求められるスキルを幅広く学び直しました。
訓練校に通っている間も、ただ授業を受けるだけでなく、個人で実際のクライアントからWebサイトの新規制作（東柏太鼓教室様）を受注したり、グループワークでAIツールを活用して厳しい納期内にサイトを完成させたりと、常に「実務」を意識して手を動かし続けてきました。
私の強みは、デザインやコーディングといった「作る」スキルと、前職で培った進行管理やクライアント対応といった「進める」スキルの両方を持っていることだと思っています。動画編集などはまだ基礎を学んだ段階ですが、新しい技術や知識を吸収していくことには抵抗がありません。 入社後は、「デザイナーだから」「コーダーだから」と自分のポジションを限定するのではなく、デザイン、実装、ディレクション、さらには動画など、その時々のチームの状況やプロジェクトに合わせて、柔軟かつ臨機応変に動ける制作者として役に立ちたいと考えています。
